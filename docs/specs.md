## 概要

SQM（Sky Quality Meter）値は、夜空の明るさを表す指標で、単位は mag/arcsec²（等級/角秒²）です。値が大きいほど空が暗い（良好な観測環境）ことを示します。

| SQM 値 | 空の状態 |
|--------|---------|
| 22.0以上 | 暗黒空（光害なし） |
| 21.0〜22.0 | 優良 |
| 20.0〜21.0 | 良好 |
| 19.0〜20.0 | 普通 |
| 19.0以下 | 光害あり |

## SQM 算出の基本原理

天体撮影画像から SQM 値を算出するには、バックグラウンド輝度を測光的に較正する必要があります。
